# PINS

A single file joe can drop a thought into and find again.

## How to use

- Add a new pin anywhere in this file. Start it with a level-2 heading: `## PIN: <short title>`.
- Pin format: title, date, what's true, where it lives (link / file / repo / email), anything that will help you find it later.
- Do not delete old pins. Append only. The whole point is that the file never gets smaller.
- Search: open the file, `Cmd+F` for a word you remember. The first match is the pin.
